Phase 1

Install the engine

One free app, installed once. ~10–15 minutes, mostly waiting.

Fintok · OpenProp
1

Find out which computer you have

This decides which download button to click — it takes 5 seconds.

On a Mac

Click the Apple logo (very top‑left corner of your screen) → About This Mac. If it says M1, M2, M3 or M4, you have "Apple Silicon." If it says Intel, you have "Intel." (Most Macs since 2021 are Apple Silicon.)

On Windows

You almost certainly want the plain Windows version — that's it.

2

Download Docker Desktop

In your web browser, go to the address below and click the blue button that matches your computer from Step 1:

↗ docker.com/products/docker-desktop
What you'll see: a file downloading — Docker.dmg on Mac, or Docker Desktop Installer.exe on Windows.
3

Install it

On a Mac

Open your Downloads folder, double‑click Docker.dmg. A window pops up — drag the whale icon onto the Applications folder shown next to it. Then open Docker from your Applications folder and click Accept / Use recommended settings.

On Windows

Double‑click the installer, click OK / Install, and let it restart your PC if it asks. Then open Docker Desktop from the Start menu and click Accept.

Don't skip the drag on Mac. Running Docker from inside the little pop‑up window causes errors — drag it into Applications first, then open it from there.
4

Wait for the whale

The first launch takes a minute or two while it sets itself up. Just wait — you don't need to click anything or make an account.

You're done when: a small whale icon 🐳 appears — in the top‑right menu bar on Mac, or the bottom‑right tray on Windows (click the ▲ arrow to show hidden icons) — and it stops moving. Leave Docker running; that's all Phase 1 needs.

Phase 2

Turn OpenProp on

Download, unzip, double‑click — it opens in your browser. ~5 min.

Fintok · OpenProp
1

Download OpenProp

When you bought OpenProp you got a download link — on the page right after you paid, and again in your email receipt. Click it and choose Save. The file lands in your computer's Downloads folder.

What you'll see: a file named openprop.zip in your Downloads folder.
2

Unzip it

Open your Downloads folder and double‑click openprop.zip. A folder named openprop appears. Drag that folder onto your Desktop so it's easy to find.

3

Double‑click "Start OpenProp"

First check that Docker is running — the whale icon 🐳 should be showing from Phase 1 (it usually starts on its own). Then open the openprop folder on your Desktop. Right there — not in any sub‑folder — is the file for your computer. Double‑click it:

Mac Start OpenProp.command  ·  Windows Start OpenProp.bat
Mac may say "unidentified developer." If it blocks the file, right‑click it → Open → Open. You only do this once, ever.
What you'll see: a black window opens and shows progress. The first time takes a few minutes while it builds your tool — this is normal. Leave the window; after this it starts in seconds.
4

Log in

Your browser opens by itself to your new tool. If it doesn't within a minute, open your browser and type this into the address bar:

↗ localhost:8787

In the Password box type the starting password, then click the green Sign in button:

Password changeme

🎉 You're in — that's the setup.

Look up any U.S. address now — location, flood zone, and income, all free. Start/stop anytime with the Start OpenProp / Stop OpenProp files; turn the page to add more.

Phase 3

Add your data — the keys

Optional. Each key unlocks more, and it's all done inside the app.

Fintok · OpenProp

🔑 How to add any key — the same 3 clicks every time

  1. In OpenProp, click Settings — the small link in the very top‑right corner.
  2. You'll see a labeled box for each key. Paste your key into the box with the matching name, then click the green Save button.
  3. On the right, the "Active now" list shows a green dot ● next to whatever you just switched on. Done — no restart, ever.
A note on "free" tiers: the free plans below are only for trying things — they cap out fast and will block you mid‑search. For real work, use the paid usage (it's small — see the costs on the next page) so you're never rate‑limited.

RentCast

Owner name, property details, tax history — the essentials
~$74/mo
for 1,000 lookups
  1. Go to app.rentcast.io/app/api and click Sign In → create an account (email or Google).
  2. Click Select Plan. For real use pick a paid plan; to just try it, the free Developer plan gives 50 lookups.
  3. Click Create API Key, name it anything, and copy the key.
  4. In OpenProp: Settings → paste into RentCast API key → Save.
⏱ ~5 min💡 Free tier is 50/mo, then it blocks — a paid plan is ~7¢/lookupKey: a long 32‑character code

Anthropic (Claude)

The AI: plain‑English search, property briefs, lead scores, outreach drafts
~$5–20/mo
pay as you go
  1. Go to console.anthropic.com and sign up (email or Google), confirming your email.
  2. Click Billing → add a card → Buy credits (the ~$5 minimum). Do this first — the key won't work without credit.
  3. Click API Keys → Create Key, then copy it right away (it's shown only once).
  4. In OpenProp: Settings → paste into Anthropic API key → Save.
⏱ ~5 min💳 A few cents per AI action; you prepay and spend down slowlyKey starts with sk‑ant‑

HUD & Census

Fair‑market rents by bedroom + neighborhood median income
Free
real gov data

HUD (rents): at huduser.gov/hudapi/public/register make a free account and tick the "Fair Market Rents" box. Create a token, then Settings → HUD FMR token.

Census (income): at api.census.gov/data/key_signup.html fill the short form, then click the activation link in the email. Then Settings → Census API key.

⏱ ~10 min for both⚠ HUD: must tick Fair Market Rents⚠ Census: must click the email link

Apify (skip trace)

Find the owner's phone number and email from a property address
~10–20¢
per lookup
  1. Go to console.apify.com/sign-up, sign up (email or Google), and click the verification email.
  2. Click your profile picture (top‑right) → Settings → API & Integrations, then copy your Personal API token.
  3. In OpenProp Settings: set Skip‑trace vendor = apify, paste the token into Skip‑trace API key → Save.
⏱ ~4 min💡 You pay only when you pull a contactToken starts with apify_api_

RealEstateAPI

Home value, equity, mortgage balance, and distress flags (foreclosure, liens)
Trial
then premium
  1. Go to realestateapi.com and click Start Free Trial; sign up with your email. You may be asked to book a short call to switch your account on — that's normal.
  2. Log in at dashboard.realestateapi.com → open the Keys section → copy your API key.
  3. In OpenProp: Settings → paste into RealEstateAPI key → Save.
⏱ ~5–10 min⚠ The one premium option — great free trial, but ~$600/mo to keep, so most people skip it
Which keys do you actually need? For most people: RentCast (the essentials) + Claude (the AI) + the free HUD/Census data. Add Apify when you want phone numbers. RealEstateAPI is only worth it if deep equity/distress data is central to your business.
