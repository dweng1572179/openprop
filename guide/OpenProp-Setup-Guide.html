Phase 1

Install the engine

Windows only — one download, one checkbox, ~3 min. Mac owners: skip this page.

Fintok · OpenProp
On a Mac? You are already done with Phase 1. macOS ships with Python — the engine OpenProp runs on — already installed. Nothing to download, nothing to click. Turn the page to Phase 2.
1

Download Python

In your web browser, go to the address below and click the big yellow Download Python button at the top. Don't worry about the version number — whichever one it offers is the right one.

↗ python.org/downloads
What you'll see: a file downloading, with a name like python‑3.13.exe, in your Downloads folder.
2

Run it — and tick the box

Open your Downloads folder and double‑click the file you just downloaded. An installer window opens with two buttons and a couple of checkboxes at the bottom.

This is the one thing you must not miss. At the bottom of that window, tick "Add python.exe to PATH" before you click anything else. It's a small checkbox and it is easy to skip — but without it, OpenProp can't find the engine and won't start.

Then click Install Now and let it run. If Windows asks "Do you want to allow this app to make changes?", click Yes.

3

Close it

When it says Setup was successful, click Close. That's the whole phase — no account, no restart, nothing to leave running.

You're done when: the installer says Setup was successful and closes. Python now sits quietly in the background forever; you will never open it or think about it again.

Phase 2

Turn OpenProp on

Download, unzip, double‑click — it opens in your browser. ~5 min.

Fintok · OpenProp
1

Download OpenProp

When you bought OpenProp you got a download link — on the page right after you paid, and again in your email receipt. Click it and choose Save. The file lands in your computer's Downloads folder.

What you'll see: a file named openprop.zip in your Downloads folder.
2

Unzip it

Open your Downloads folder and double‑click openprop.zip. A folder named openprop appears. Drag that folder onto your Desktop so it's easy to find.

3

Double‑click "Start OpenProp"

Open the openprop folder on your Desktop. Right there — not in any sub‑folder — is the file for your computer. Double‑click it:

Mac Start OpenProp.command  ·  Windows Start OpenProp.bat
Mac may say "unidentified developer." If it blocks the file, right‑click it → Open → Open. You only do this once, ever.
What you'll see: a black window opens and shows progress. The first time takes a few minutes while it builds your tool — this is normal. Leave the window; after this it starts in seconds.

Phase 3

Add your data — the keys

Optional. Each key unlocks more, and it's all done inside the app.

Fintok · OpenProp

🔑 How to add any key — the same 3 clicks every time

  1. In OpenProp, click Settings — the small link in the very top‑right corner.
  2. You'll see a labeled box for each key. Paste your key into the box with the matching name, then click the green Save button.
  3. On the right, the "Active now" list shows a green dot ● next to whatever you just switched on. Done — no restart, ever.
One dropdown decides everything: "Primary property provider." It's on the same Settings page, and it must match the key you pasted. OpenProp ships set to reapi (RealEstateAPI). If you went with RentCast, change it to rentcast and hit Save — otherwise your key sits there unused and searches come back empty.
A note on "free" tiers: the free plans below are only for trying things — they cap out fast and will block you mid‑search. For real work, use the paid usage (it's small — see the costs on the next page) so you're never rate‑limited.

RentCast

Owner, property details, tax history — best for one address at a time
~$74/mo
for 1,000 lookups
  1. Go to app.rentcast.io/app/api and click Sign In → create an account (email or Google).
  2. Click Select Plan. For real use pick a paid plan; to just try it, the free Developer plan gives 50 lookups.
  3. Click Create API Key, name it anything, and copy the key.
  4. In OpenProp: Settings → paste into RentCast API key → set Primary property provider to rentcast → Save.

Which keys do you actually need? It comes down to one question — are you looking up addresses, or building lists?
Looking up one address at a time: RentCast + Claude + the free HUD/Census data. Set the provider dropdown to rentcast. Add Apify when you want phone numbers.
Searching a whole zip and filtering by equity or distress: that's RealEstateAPI, and only RealEstateAPI. It's the one provider that returns owner, value, equity and foreclosure/lien flags on every row — a RentCast‑backed area search comes back mostly blank, so those filters match nothing. Use the free trial to see if list‑building is your business before you commit to the price.
